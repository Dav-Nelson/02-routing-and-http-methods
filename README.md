# 01 HTTP Server

A simple Express server built to understand how HTTP servers work.

## Learning Objectives

- Understand Node.js
- Understand Express
- Understand Ports
- Understand Processes

## Technologies

- Node.js
- Express
- TypeScript

## How to Run

1. Clone repository
2. npm install
3. npx tsx src/index.ts
4. Visit localhost:3000
